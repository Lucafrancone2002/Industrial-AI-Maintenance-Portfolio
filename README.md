# Industrial AI & Automation Portfolio
### Luca Francone - Automation & Control Engineering

This repository contains technical implementations of AI and Signal Processing techniques applied to industrial maintenance. My work bridges the gap between **Field Service Engineering** and **Advanced Control Theory**.

## 🚀 Projects Overview

### 1. Fleet-Scale Remaining Useful Life (RUL) Prediction
* **Target:** Reducing unscheduled downtime by predicting equipment failure.
* **Dataset:** NASA CMAPSS (Turbofan Engine Degradation).
* **Key Skills:** Time-series analysis, Feature Engineering, Regression Models (Random Forest/XGBoost).

### 2. Anomaly Detection in Control Signals
* **Target:** Identifying sensor drift and hardware degradation in automated loops.
* **Methodology:** Using Autoencoders and Residual Analysis to isolate faults before system failure.
* **Key Skills:** Digital Signal Processing, Deep Learning (PyTorch/TensorFlow), Control Theory.
